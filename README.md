# Preprocessor stub

## Installation

Install node.js

Run
```bash
# install gulp global
npm i -g gulp

# go to project dir
cd %project dir%

# install project node modules
npm i
```

## Work with sass

Copy images from src to dist

```bash
gulp images
```

```bash
gulp images:watch
```


## Work with sass

Compile sass

```bash
gulp sass
```

Run watcher
```bash
gulp sass:watch
```

## Common work

build project

```bash
gulp build
```

watch all

```bash
gulp watch
```
